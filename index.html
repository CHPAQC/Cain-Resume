project

  • 微信小程序———鹏城翻译

    2018
    • 鹏城翻译是一款翻译语言的小工具，在移动端使用，可以再语言页面选择所需翻译的语种，并且翻译过后会在历史页面有遗留，而且点击历史记录的所需翻译会跳转到翻译页面翻译当前语种，方便下次查找，整个工具主要是根据小程序技术文档来写的。
    • 查看效果可以在微信小程序处搜索鹏城翻译，或者扫描小程序码
    • 技术： 百度api + iconfont + JavaScript + Promise + md5加密 + WXML + WXSS + wx.request + wx.showToast + wx.switchTab
    • 这次项目作为深入实践，遇到不少问题，细节分析请移步： 个人博客
    • 在这个项目过程中，由于没有后台，且需要一个翻译api，所以最后选择了百度翻译api，在这个过程中我看了百度翻译api使用文档，其中需要用到md5加密，然后就是在项目中，比较关键的一点是无论在语言页面还是历史页面都需要有一个缓存的数据，然后就用到了wx.setStorageSync、wx.getStorageSync这两个方法。

    Cain版网易云音乐

    2018
    • 这个项目是用jQuery实现的仿网易云音乐，有播放歌曲、切换、暂停、搜索等功能，PC 端歌曲上传、删除、修改等功能。使用 jQuery、MVC，以及七牛、和自己用json作为数据库实现。使用vConsole进行调试。
    • 技术：MVC、Observer设计模式、jQuery、响应式布局 、iconfont、svg、七牛&json作为数据库
    • 项目中，因为需要一个后台储存歌曲，所以我想到自己写json配合七牛充当后台，用文件名充当接口获取数据，然后渲染。在css过程中遇到margin合并问题，还学会了一个新的办法（noCollapse）来解决，在搜索页面用到promise，通过filter返回数组中的一个子集，最后在遍历返回子集渲染成所需要的数据放在页面上。
    • JS Try it Github 移动端请扫码
